# Kids Clothing E-Commerce Site

## Setup
1. Install dependencies in backend and frontend.
2. Start MongoDB.
3. Run backend: `npm start` in backend/.
4. Run frontend: `npm start` in frontend/.
5. Add sample products via POST to /api/products (e.g., {"name": "T-Shirt", "price": 20, "category": "boys", "image": "url"}).

## Features
- Product listings by category.
- User auth.
- Cart and Stripe checkout.
